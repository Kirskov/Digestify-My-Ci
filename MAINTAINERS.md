# Maintainers

## Roles

**Maintainer**
- Reviews and merges pull requests
- Triages issues and bug reports
- Publishes releases and signs artifacts
- Manages repository settings, secrets, and access
- Sets project direction and priorities

**Contributor**
- Submits pull requests and bug reports
- No access to sensitive resources or release tooling
